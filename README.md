# Bossy Skincare

Bossy Skincare es una página estática hecha en React y TailwindCSS

## Deployment

To deploy this project run

```clone
  git clone https://github.com/FrancoSaldano/BOSSY.git
  cd BOSSY
  npm install
  npm run dev
```
